This Web Scraping program is specifically built to download images from the dragonball.fandom website.

Requirements:
Have Node.js installed in your computer.

How to use:
1.- Run "npm install puppeteer" in the terminal (inside this project folder), this should create a folder named "node_modules".
2.- Run "node index" in the terminal (inside this project folder), this should start the web scraping and get you the images downloaded 1 by 1 inside the project folder.

If having any problem:
1.- Delete the next files from the project folder:
a) package-lock.json.
b) package.json.
2.- Follow the instructions of "How to use" section in order.
